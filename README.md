# 現場派工協作所

依現場即時狀況動態分派工單至設備或人員，確保排程落地執行。

- 原始系統：現場派工系統（Real-Time Dispatching, RTD）
- 產業分類：智慧製造｜製造管理
- 使用部門：現場班組長、生管、製造工程
- 工作流程：排程下達 → 現場狀態擷取 → 動態派工 → 執行回報 → 異常重派
- AI 應用：AI依歷史數據預測最佳派工組合、智慧偵測異常延遲並主動建議調整
- 主要 KPI：派工達成率、平均等待時間、產線平衡度
- 代表產品：多為MES模組延伸功能，如Siemens Opcenter Execution、Critical Manufacturing MES
